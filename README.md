# X Screen Time Limiter

A Chrome extension to limit your time spent on X (Twitter).

## Installation

1. Open Chrome and navigate to `chrome://extensions/`
2. Enable "Developer mode" (toggle in the top right)
3. Click "Load unpacked"
4. Select the extension directory

## Usage

### Setting Your Daily Limit

1. Click the extension icon in Chrome toolbar
2. Enter your daily time limit in minutes (default: 60)
3. Click "Save"

### While Browsing X

- A timer appears in the bottom-right corner showing remaining time
- The timer counts down every second
- When time runs out, the page background turns red
